# Blog - template
Simple responsive blog template created with React.js, Sass
## Table of contents
* [General info](#general-info)

## Technologies
Project is created with:
* React.js
* ECMAScript 6
* CSS3/SASS
* HTML5
## Setup
To run this project, install it locally using npm:

```
$ npm install
$ npm start
```
